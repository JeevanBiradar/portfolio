# Jeevan Biradar Portfolio

Personal portfolio website for Jeevan Biradar, focused on Java full stack development, projects, experience, and contact details.

## Stack

- HTML
- Tailwind CSS (local build)
- Custom CSS
- Vanilla JavaScript

## Local setup

1. Install dependencies:
   `npm install`
2. Rebuild Tailwind CSS after class changes:
   `npm run build:css`
3. Open `index.html` in the browser.

## Project files

- `index.html` - main site markup
- `styles.css` - custom styles and interactions
- `script.js` - scroll, nav, and mobile menu behavior
- `tailwind.config.js` - local Tailwind config
- `tailwind.input.css` - Tailwind entry file

## Notes

- The generated `tailwind.css` is built locally from Tailwind.
- Fonts and Material Symbols are currently loaded from Google Fonts.
